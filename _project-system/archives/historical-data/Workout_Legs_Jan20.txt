TUESDAY: LEG DAY (Volume & TUT Focus)
---------------------------------------
Context: 44-Hour Fasted State
Goal: Maximum Glycogen Depletion & Glute/Quad Hypertrophy
Safety Alert: Orthostatic Hypotension (Dizziness). STAND UP SLOWLY!

0. PRE-WORKOUT PROTOCOL (Critical)
   - t-45 mins: Drink Salt + Citrulline (12g) + Creatine (5g).
   - t-15 mins: Start sipping EAA (20g).
   - Warm-up: 5-10 mins brisk walking.

1. Barbell Hip Thrusts (PRIORITY #1)
   Goal: Wake up glutes & protect lower back.
   - Warm-up: 15 reps @ 40 kg (or empty bar + 20kg). Hold 2 sec at top.
   - Feeder Set: 12 reps @ 60 kg.
   - Working Set 1: 10-12 reps @ 80-90 kg.
   - Working Set 2: 10-12 reps @ 80-90 kg.
   *Technique: Chin tucked, drive through heels. Hard squeeze at top.

2. Leg Extensions
   Goal: Isolate quads & pre-exhaust.
   - Set 1: 20 reps @ 30 kg (Warm joints).
   - Set 2: 15 reps @ 40 kg.
   - Set 3: 12-15 reps @ 45-50 kg (Control descent!).
   - Set 4: 12-15 reps @ 45-50 kg.

3. Leg Press (Wide Stance)
   Goal: Mass builder. CAUTION: Dizziness risk.
   - Set 1: 15 reps @ 80-100 kg.
   - Set 2: 12 reps @ 140 kg.
   - Set 3: 10-12 reps @ 160-170 kg. (Tempo: 3 sec down).
   - Set 4: 10-12 reps @ 160-170 kg.
   - Set 5 (Drop Set): 15-20 reps @ 120 kg.
   *Safety: Sit for 15 sec after each set before standing up!

4. Goblet Squat (or Hack Squat)
   Goal: Depth & stretch.
   - Set 1: 12 reps @ 20 kg (DB) / Sled only.
   - Set 2: 10 reps @ 30 kg (DB) / +20kg Machine.
   - Set 3: 10 reps @ 30-35 kg (DB) / +40kg Machine.
   - Set 4: 10 reps @ 30-35 kg (DB) / +40kg Machine.

5. Lying Leg Curls
   Goal: Hamstring isolation.
   - Set 1: 15 reps @ 25 kg.
   - Set 2: 12 reps @ 35 kg.
   - Set 3: 10-12 reps @ 40 kg.
   - Set 4: 10-12 reps @ 40 kg.

6. Adductor Machine
   Goal: Inner thigh volume.
   - Set 1: 15 reps @ 40 kg.
   - Set 2: 12-15 reps @ 50-55 kg.
   - Set 3: 12-15 reps @ 50-55 kg.

7. Calves (Standing or Leg Press)
   - 4 Sets x 15 Reps (Heavy).
   - Full stretch at bottom, hold at top.

POST-WORKOUT CHECKLIST
----------------------
1. Immediately: Finish EAA water.
2. +15 mins: Drink 12.5g Collagen + 1g Vitamin C.
3. +60 mins: THE REFEED MEAL (250g Chicken + 200g Rice + Salt + Olive Oil).
